Easy Writing, or, How I Learned to Stop Worrying and Love Markdown

11.01.2012

About six months ago, I started using Markdown to take notes in class, to write essays, to keep track of meetings, to make project to-do lists — for just about everything. Markdown is the reason I chose to use Scriptogr.am for this blog. People who see my screen in a class often ask what I‘m doing, and I’ve been recognized elsewhere as “the kid who codes his notes.” It's really not that scary. I think Markdown is for everyone. So this is an explainer on how I switched my written life to Markdown, and an argument for why more people, especially journalists, should do so.
